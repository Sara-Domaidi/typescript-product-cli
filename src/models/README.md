# Product Management CLI Tool

A simple Command Line Interface (CLI) application built with **TypeScript**, **Sequelize ORM**, and **PostgreSQL** to manage product records.

## Features
- **Add** new products.
- **List** all products.
- **Get** a specific product by ID.
- **Update** product details.
- **Delete** products.

## Tech Stack
- **Language:** TypeScript
- **Database:** PostgreSQL
- **ORM:** Sequelize
- **CLI Framework:** Commander.js
